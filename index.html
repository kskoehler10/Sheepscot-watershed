About this project ►

The flow-distance raster was derived from a 10 m resolution LiDAR-based DEM using GRASS GIS r.watershed. After filling sinks in the DEM, D8 flow directions trace the least-cost downhill path from every pixel to the watershed outlet at the Sheepscot estuary near Wiscasset, ME. Each cell's value is the cumulative overland-flow distance to that outlet.

The stream network was extracted with r.stream.extract and ordered using the Strahler system: headwater channels are order 1; the Sheepscot River mainstem reaches order 8. Line widths scale with order, reflecting the accumulation of upstream drainage area.

Waterbodies are from the NHD (National Hydrography Dataset, HUC 01050003). Roads, trails, and named waterways are from OpenStreetMap. The viridis color ramp on the flow-distance layer is applied client-side via the Mapbox GL JS raster-color paint property.
